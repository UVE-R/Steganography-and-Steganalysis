Steganography Programs:
embed_msg.py - An application to allow the user to embed a message into a given
               image, or choose to extract a message from an image

LSBClass.py  - Contains the code for LSB embedding and matching separated into classes

primes.py    - A set of helper functions for generating large prime numbers


Steganalysis Programs:
stat_attack.py - An application to allow the user to select the RS or SPA attack
		 to analyse a given image
SPA.py         - Contains the code for the SPA attack
RS.py          - Contains the code for the RS attack


training.ipynb - The training script used for the CNN
model.keras    - The saved CNN model from the 9th epoch
